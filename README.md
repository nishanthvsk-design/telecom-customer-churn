# Telecom Customer Churn Dashboard

An interactive R Shiny app to analyze Telco customer churn by contract type, internet service, tenure, and monthly charges.  
Empowers users to visualize churn patterns and discover actionable insights for customer retention strategies.

---

## Features

- **Dynamic dashboards**: Explore churn by customer attributes.
- **Modern UI**: Responsive, Bootstrap-themed layout.
- **Live filtering**: Slice and dice cohorts interactively.
- **Visualizations**: Download high-resolution PNG plots.
- **Data cleaning & transformation**: Built with tidyverse for robust analysis.
